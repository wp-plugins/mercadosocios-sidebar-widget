Add items from MercadoLibre to your Wordpress sidebar using your affiliate. 

== Description ==

Displays a gallery of MercadoLibre´s products with affiliate code inserted according to your country, category and keywords. 

== Installation ==

1. Upload `mercadosocios.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget like any other widget.

== Frequently Asked Questions ==

1. Where can I find the id of each category? 
Try http://www.mercadolibre.com.ar/jm/categsXml?as_site_id=MLA&as_max_level=3

2. What is the country code for...?
MLB br
MLA ar
MLM mx
MLC cl
MLV ve
MCO co

3. Where can I find the Afiliado?

Login into MercadoSocios and create any link personalizado. You will obtain something like http://pmstrk.mercadolibre.com.ar/jm/PmsTrk?tool=000000&go=

Your Afiliado is the number right after ?tool= In this case: 000000
